📊 AI Review Intelligence Dashboard
A modern, responsive web application designed for sellers to monitor and analyze customer feedback across multiple platforms like Amazon, Flipkart, and Playstore. This dashboard transforms raw review data into actionable insights using dynamic filtering and interactive charts.

🚀 Features
Real-time Analytics Cards: Instantly view Total Reviews, Sentiment Breakdown (Positive/Negative), and Average Ratings.

Interactive Data Visualization: Integrated Chart.js to provide a visual breakdown of rating distributions (1-5 Stars).

Multi-Platform Filtering: Filter reviews by source (Amazon, Flipkart, Website, Playstore) and specific Star Ratings.

Dynamic Data Table: A clean, structured overview of customer names, feedback text, and automated sentiment labels.

Responsive Dark UI: Built with a "Modern SaaS" aesthetic using CSS Flexbox and Grid, optimized for readability.

🛠️ Tech Stack
Frontend: HTML5, CSS3 (Custom Properties, Grid, Flexbox)

Scripting: Vanilla JavaScript (ES6+)

Library: Chart.js for data representation

Fonts: Poppins via Google Fonts
